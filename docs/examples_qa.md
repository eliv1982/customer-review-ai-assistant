# Примеры отзывов и ожидаемого результата анализа

Все примеры **вымышлены**: имена, товары и ситуации не связаны с реальными лицами. Используются для демонстрации формата выхода модели (`sentiment`, `topic`, `summary`, `reply_draft`). Фактический ответ API может слегка отличаться по формулировкам при повторном запросе.

---

## Пример 1

| Поле | Значение |
|------|----------|
| **Текст отзыва** | Курьер привёз коробку мокрой, внутри крошки пластика. Техника работает, но впечатление испорчено. |
| **Тональность** | `negative` |
| **Тема** | `delivery` или `packaging` (в зависимости от акцента модели) |
| **Краткая сводка** | Клиент недоволен состоянием упаковки и доставки; товар функционирует, но есть претензия к сервису привоза. |
| **Черновик ответа** | Здравствуйте! Приносим извинения за состояние упаковки при доставке — так быть не должно. Пожалуйста, укажите номер заказа, мы проверим ситуацию и свяжемся с вами с обратной связью. |

## Пример 2

| Поле | Значение |
|------|----------|
| **Текст отзыва** | Спасибо за быструю доставку и вежливого оператора! Всё объяснили, кофе отличный. |
| **Тональность** | `positive` |
| **Тема** | `delivery` / `product_quality` |
| **Краткая сводка** | Клиент благодарит за скорость доставки, работу поддержки и качество продукта. |
| **Черновик ответа** | Благодарим за тёплые слова! Нам очень приятно, что доставка и консультация прошли комфортно. Будем рады видеть вас снова. |

## Пример 3

| Поле | Значение |
|------|----------|
| **Текст отзыва** | Предлагаю добавить в личный кабинет напоминание о замене фильтра — так проще планировать обслуживание. |
| **Тональность** | `neutral` |
| **Тема** | `website` или `other` |
| **Краткая сводка** | Клиент предлагает улучшение цифрового сервиса (напоминания), без жалобы. |
| **Черновик ответа** | Спасибо за идею! Мы передали её команде, отвечающей за развитие сервиса. Обратная связь очень ценна при планировании обновлений. |

## Пример 4

| Поле | Значение |
|------|----------|
| **Текст отзыва** | На сайте было «в наличии», ждала две недели. Поддержка отвечала шаблонами, номер заказа не помогает. |
| **Тональность** | `negative` |
| **Тема** | `website` / `support` |
| **Краткая сводка** | Клиентка недовольна расхождением статуса на сайте со сроками и качеством ответов поддержки. |
| **Черновик ответа** | Здравствуйте! Приносим извинения за ожидание и ощущение шаблонных ответов — это важный сигнал для нас. Пожалуйста, продублируйте номер заказа в ответном сообщении, мы передадим кейс ответственной линии для проверки. |

## Пример 5

| Поле | Значение |
|------|----------|
| **Текст отзыва** | Цена нормальная, но прибор шумит сильнее, чем ожидал. В целом доволен, упаковка с вмятиной, но товар цел. |
| **Тональность** | `mixed` |
| **Тема** | `product_quality` / `price` / `packaging` |
| **Краткая сводка** | Смешанные впечатления: устраивает цена и целостность товара, не устраивает шум и вид упаковки. |
| **Черновик ответа** | Здравствуйте! Спасибо за честную обратную связь. Нам жаль, что уровень шума и вид упаковки разочаровали. Если удобно, опишите, пожалуйста, режим работы прибора — мы подскажем дальнейшие шаги по консультации. |

## Пример 6

| Поле | Значение |
|------|----------|
| **Текст отзыва** | Доставка в срок, но приложение к роботу-пылесосу часто вылетает. Надеюсь, почините в обновлении. |
| **Тональность** | `mixed` или `neutral` |
| **Тема** | `delivery` + `website` / `product_quality` (ПО как часть продукта) |
| **Краткая сводка** | Клиент отмечает своевременную доставку и проблемы со стабильностью мобильного приложения. |
| **Черновик ответа** | Здравствуйте! Рады, что доставка прошла в срок. По приложению: приносим извинения за неудобства, информацию передадим в техническую поддержку продукта для анализа. Если можете, укажите модель и версию приложения — это ускорит проверку. |
